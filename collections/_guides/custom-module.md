# Custom Breeze-friendly Module

Module directory structure:

```
<Vendor>_<Module>/
├── view/frontend/web/
│   ├── css/
│   │   ├── breeze/
│   │   │   ├── _default.less
│   │   │   ├── _checkout.less
│   │   ├── _breeze-extend.less
│   ├── js/
│   │   ├── breeze/
├── layout/
├── templates/
```
